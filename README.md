# Tasty Table

A modern, responsive restaurant landing page built with HTML, CSS, and JavaScript.

## Live demo

https://rudrashivani95-lab.github.io/Food-Fusion/

## Features

- Responsive navigation menu for desktop and mobile
- Hero section with call-to-action buttons
- Menu cards with elegant styling
- Services, testimonials, and contact sections
- Light/dark mode toggle
- Form validation for the contact form
- Scroll-to-top button

## How to run

Open index.html in your browser, or serve the folder with a simple static server.

Example:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
